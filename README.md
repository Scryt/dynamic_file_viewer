# pdf_viewer

## Project setup
```
npm install
```
Requires .env file:
For easy startup create .env file in project directiory with given data inside:

```
PORT=8080
HOST=127.0.0.1
VUE_APP_API_ROOT=http://localhost:8080
```

### Star server / vue instanace
```
npm start server.js
npm run serve
```
